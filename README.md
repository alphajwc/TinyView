## Features
* Zoom In
* Zoom Out
* Left Rotation
* Right Rotation
* Move Image

## Getting Started

### Installation

NPM
```
npm install tinyview.js
```
In browser:
```
<link  href="/path/to/tinyview.css" rel="stylesheet">
<script src="/path/to/tinyview.js"></script>
```
